A React/Redux Web application to show movie demographics from iMDB as well as ratings from Metacritic, Flixster, and Rotten Tomatoes.

## Tech Stack
* React.js (JSX)
* Redux
* Javascript
* Apis: themoviedb.org, omdbapi.com, imdb.
