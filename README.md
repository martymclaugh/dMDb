A React/Redux Web application to show movie demographics from iMDB along with ratings from Metacritic, Flixster, and Rotten Tomatoes.

## Tech Stack
* React.js (JSX)
* Redux
* JavaScript
* Apis: themoviedb.org, omdbapi.com, imdb, YouTube.
* Chart.js
